Navigation

Desktop navigation



This is the logged-in state of the navigation bar. If no one is logged in, the "My Account" dropdown does not appear.




This is the category dropdown menu. The blue squares are category icons. Note, this is the logged-out state.




Breadcrumbs appear on desktop and tablet.




This new sidebar navigation style is for the devHub only.


Mobile Navigation




Here we have two versions of the mobile navigation bar. The top two are the two different states of the standard mobile Marketplace app: browse and settings. The third navigation bar is specific to Tarako devices. There is no logged in state for this fork of the Marketplace.


Events

Events appear at the top of the screen, pushing the page below down. When dismissed by tapping 'x', the event block slides up off the page.
